# Product Overview

## Habit Tracker Windows Desktop Application

A native Windows desktop application for habit tracking that allows users to:
- Add new habits to track
- Navigate through months to view habit progress
- Mark habits as completed for specific days
- Maintain a visual record of habit consistency
- View weekly statistics and analytics
- Switch between light and dark themes

The application is built with React and packaged as a native Windows application using Electron, providing a seamless desktop experience with native menus, keyboard shortcuts, and system integration.
